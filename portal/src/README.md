# A-Frame: Image Target Portal

This example shows off the popular portal illusion in web AR using xrextras-hider-material 
and an image target.

![](https://media.giphy.com/media/kBxbQNBBoB4xtPAG8h/giphy.gif)

Point your phone at this image target to test:

![](https://cdn.8thwall.com/web/img/readme/wall-1.jpg)

### Project Components

```image-target-portal``` opens portal and animates the ball inside when an image target is detected.

- name: the name of the image target as it appears on the "Image Targets" page

```xrextras-hider-material``` is applied to any mesh or primitive that must be transparent while 
blocking the rendering of models behind it.
